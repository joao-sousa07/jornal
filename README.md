# Jornal Diário

Um sistema simples de notícias com comentários desenvolvido em Django.

## Funcionalidades

- Listagem de notícias
- Visualização de notícias completas
- Sistema de comentários nas notícias

## Tecnologias Utilizadas

- Python 3.14.3
- Django 6.0.4
- SQLite3
- HTML5/CSS3
